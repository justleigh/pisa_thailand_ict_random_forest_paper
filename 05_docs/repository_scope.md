# Repository scope

This repository is provided to support transparency and auditability for the published paper on ICT-related, contextual, and behavioural predictors of student achievement in Thailand using PISA 2022.

## Purpose

The repository documents the analysis workflow underlying the published article and provides selected scripts, outputs, and public-facing metadata directly relevant to the paper’s reported findings. Its purpose is to help readers understand how the published analysis was structured and how the main synthesis results were produced.

## What is included

The repository includes:

- selected preparation scripts used in the published workflow
- the random forest scripts used in the paper
- the synthesis scripts and selected synthesis outputs underpinning the article’s main claims
- a reduced public-facing inventory of variables used in the published analysis
- documentation describing the scope of the repository and the broad structure of the analytical workflow

## What is not included

This repository does not reproduce the full internal PhD project workflow. In particular, it does not include:

- the full private variable-mapping table
- detailed framework assignment files
- private cleaned or imputed datasets
- intermediate data products generated during the broader internal workflow
- internal tracking files, draft materials, or private workflow resources
- the full inferential modelling pipeline planned for later stages of the PhD project

## Public-private boundary

The public materials included here are intended to support understanding of the published paper while maintaining a clear boundary around selected internal research assets. Some project-specific materials, especially the full variable-mapping system and associated preparation resources, represent substantial intellectual and procedural work developed as part of a broader research programme and are therefore not publicly released.

## Reproducibility note

This repository should be understood primarily as a transparency and auditability resource rather than as a fully self-contained end-to-end reproduction package. The scripts included here reflect the implemented workflow used in the study, but some of them depend on intermediate data objects, private metadata resources, or local adaptations that are not publicly released. As a result, the repository is intended to document and support inspection of the published workflow rather than to provide a turnkey rerun of the complete internal pipeline.

## Relation to the published paper

The scripts and outputs included here correspond specifically to the published random forest paper based on the Thailand subset of PISA 2022. The repository is therefore paper-specific and should not be interpreted as a complete archive of the broader PhD project.
