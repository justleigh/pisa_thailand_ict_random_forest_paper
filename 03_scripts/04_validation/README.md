# Validation scripts

This folder contains the validation scripts used to support the manuscript's random forest validation and sensitivity checks.

The validation workflow covers:
- predictive performance checks using OOB R² and RMSE
- rank-stability checks across plausible values
- sensitivity checks using alternative random forest specifications and permutation importance
- correlation diagnostics for highly correlated predictor families
- summary table generation for the manuscript and appendix

The scripts may require the cleaned analytical dataset and private project metadata, which are not redistributed in this public repository.
