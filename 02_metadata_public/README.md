# Public metadata

This folder contains a reduced public-facing inventory of variables used in the published paper on ICT-related, contextual, and behavioural predictors of student achievement in Thailand using PISA 2022.

## File included

- `public_variable_inventory.csv`  
A high-level inventory of variables used in the published analysis, including variable names, public-facing labels/descriptions, source questionnaire, and broad analytical role.

## Purpose

This file is provided to support transparency regarding the variables included in the published paper and their general role in the analysis.

## What is not included

This folder does not reproduce the project’s full private variable-mapping table, detailed framework assignment system, transformation logic, or proprietary metadata resources. Those materials were part of the broader internal research workflow and are not publicly released.

## Note

The public metadata provided here is intended to support understanding of the published paper rather than to reproduce the full internal research pipeline.
