# Validation outputs

This folder contains selected public validation outputs corresponding to the manuscript's main validation summary and supplementary validation tables.

These files support:
- Table 2: Validation summary for exploratory random forest screening models
- Table 10: Predictive performance and rank stability by model family
- Table 11: Sensitivity checks by validation level and comparison
- Table 12: Correlation diagnostics by validation level

Large intermediate audit files and private working outputs are not redistributed. The scripts used to generate the validation workflow are included in `03_scripts/04_validation/`.
