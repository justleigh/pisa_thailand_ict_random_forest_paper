# Data access

This folder documents the input data requirements for the public repository.

The published paper is based on the Thailand subset of PISA 2022. Raw PISA data are not redistributed in this repository. Users who wish to run the public scripts should obtain the relevant PISA 2022 data from the official OECD/PISA source and place the required input file(s) in the local folder structure expected by the scripts.

## Expected input

The public scripts are designed for use with the PISA 2022 Thailand data used in the published analysis. If local filenames or paths differ, users should update the relevant script settings accordingly.

## Important note

This repository is intended to support transparency for the published paper. It does not provide the original raw data and does not reproduce the full private project workflow.
